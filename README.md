## Features

- **Theme Mode Switcher**: The site allows users to switch between a dark mode and a light mode to enhance user experience in different lighting conditions.
- **User Interface**: Simple and elegant interface design that facilitates easy navigation.
- **Data Management**: Data management tools such as add, search, and delete functionalities.

## Technologies Used

- **HTML**: For structuring the web pages.
- **CSS**: For styling the user interface.
- **JavaScript**: For adding interactivity to the pages and implementing functions like theme switching.

## How to Use

Step-by-step instructions on how to use the website, from accessing the homepage to performing various functions such as adding new data or searching.

### Switching Themes

1. Click on the theme switch button in the top corner.
2. Choose between dark or light mode.

### Adding Data

1. Fill in the required fields in the form.
2. Press 'submit' to save the data.

### Searching

1. Type in the search bar.
2. Click 'search' to display the results.
